Get started with Brokle in just a few minutes.

## Installation

Install the Brokle SDK using your preferred package manager.

### Python

```bash
pip install brokle
```

### JavaScript

```bash
npm install brokle
```

## Initialize the Client

Create a new Brokle client with your API key.

```python
from brokle import Brokle

client = Brokle(api_key="your-api-key")
```

## Create Your First Trace

Start tracing your AI application calls.

```python
with client.start_as_current_span("my-first-trace") as span:
    # Your AI code here
    response = openai.chat.completions.create(...)
    client.update_current_span(output=response.choices[0].message.content)
```
